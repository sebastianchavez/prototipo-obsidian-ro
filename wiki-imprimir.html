Versión para imprimir

Esta herramienta genera una versión optimizada de cualquier página de la wiki, ideal para imprimir en papel o guardar como PDF. La versión para imprimir elimina elementos innecesarios como menús, barras laterales y navegación.

Cómo usar esta herramienta

Paso 1: Introduce el nombre de la página que deseas imprimir en el campo de búsqueda.
Paso 2: Se abrirá una nueva ventana con la versión optimizada. Usa Ctrl+P (Windows) o Cmd+P (Mac) para imprimir.

Características de la versión para imprimir

  • Eliminación de elementos UI: Se remueven menús, barras laterales, pestañas de navegación y otros elementos de interfaz.
  • Optimización de espacio: El contenido se ajusta para ocupar todo el ancho de la página.
  • Estilos simplificados: Se usan colores e formatos aptos para impresión en blanco y negro.
  • Enlaces como texto: Los enlaces se muestran como texto plano con la URL entre paréntesis.
  • Imágenes ajustadas: Las imágenes se redimensionan para caber en la página impresa.

Consejos para imprimir

Consejo Descripción
Configuración de página Usa márgenes pequeños y orientación vertical para aprovechar mejor el espacio.
Impresión a doble cara Activa la impresión a doble cara si tu impresora lo permite, para ahorrar papel.
Guardar como PDF En lugar de imprimir directamente, selecciona "Guardar como PDF" como destino para tener una copia digital.
Seleccionar páginas Si solo necesitas partes específicas, usa la opción de imprimir páginas seleccionadas.
Calidad de impresión Para documentos de referencia, usa calidad "Borrador" para ahorrar tinta.

Páginas populares para imprimir

Estas son algunas de las páginas más solicitadas para imprimir:

  • Guía Básica - Guía completa para nuevos jugadores.
  • Reglas - Normas del servidor para consulta rápida.
  • Builds - Builds recomendadas para cada clase.
  • MVPs - Tabla de MVPs con sus estadísticas y recompensas.
  • Rates - Tasas del servidor y eventos activos.
Nota: La versión para imprimir se actualiza automáticamente con cada edición de la página. Si necesitas una versión actualizada, simplemente vuelve a generar la versión para imprimir.
